# Smart Brain
Facial recognition app that uses machine learning to identify faces within photos, includes user ranking system and encrypted user log-in/registration.

Technologies:

- React + Redux
- Clarifai API
- bCrypt
- PostgreSQL
- Deployed on Heroku
